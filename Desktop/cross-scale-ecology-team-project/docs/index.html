土壤有机质分子特征与细菌丰富度可复现分析

课程：数据驱动研究方法（D2RS）| 结课作业

一、项目简介

本项目基于 Nature Food (2025) 文献，使用 R 语言实现可复现分析，探究长期农田土壤有机质（SOM）分子多样性、热力学稳定性与细菌丰富度之间的关系，验证核心结论：

细菌丰富度通过调控分子多样性–热力学稳定性权衡关系，提升土壤有机质热稳定性。

二、研究背景

  • 土壤有机质稳定性决定土壤固碳、肥力与粮食安全
  • 微生物是有机质形成与稳定的核心驱动因子
  • 长期耕作下 SOM 分子演化机制仍需深入解析

三、研究目标

  1. 分析长期耕作下 SOM 分子特征的时间变化规律
  2. 验证分子多样性与热力学稳定性的权衡关系
  3. 揭示细菌丰富度对 SOM 稳定性的调控作用
  4. 构建结构清晰、可复现、可部署的科研分析流程

四、核心结论

  1. SOM 分子热力学稳定性与整体热稳定性显著正相关
  2. 长期耕作下，分子多样性与热力学稳定性呈显著负向权衡
  3. 细菌丰富度越高，越能促进 SOM 向更稳定方向演化
  4. 提高土壤细菌丰富度有利于农田固碳与土壤可持续利用

五、可复现性说明

本项目使用 R + renv 实现环境锁定，代码、数据、结果完全开放，支持一键复现所有图表与统计结果。
