# 土壤有机质分子特征与细菌丰富度可复现分析
D2RS 2026 结课项目

## 项目简介
本项目基于 Nature Food (2025) 文献，复现长期稻田与旱地农田土壤有机质（SOM）的分子演化规律，验证：
**细菌丰富度通过调控分子多样性–热力学稳定性权衡，提升SOM热稳定性**。

项目满足：
- 结构清晰
- 完全可复现
- 可一键部署 GitHub Pages
- 内容规范、符合课程要求

## 研究内容
1. 长期耕作下 SOM 分子多样性变化
2. 分子热力学稳定性与热稳定性的协同关系
3. 分子多样性与热力学稳定性的长期权衡
4. 细菌丰富度对上述过程的调控作用

## 复现步骤
1. 打开 RStudio 项目 wzfx.Rproj
2. 恢复环境
install.packages("renv")
renv::restore()
3. 运行 code/som_analysis.Rmd
4. 自动生成所有图表与报告
