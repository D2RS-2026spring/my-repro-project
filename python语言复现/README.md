# 土壤有机质分子特征与细菌丰富度可复现分析
D2RS 2026 结课项目

---

## 项目简介
本项目基于 Nature Food (2025) 文献，使用 Python 复现长期稻田与旱地农田土壤有机质（SOM）的分子演化规律，验证：**细菌丰富度通过调控分子多样性-热力学稳定性权衡，提升SOM热稳定性**。

项目满足：
- 结构清晰
- 完全可复现
- 可一键运行所有图表
- 内容规范、符合课程要求

---

## 研究内容
1. 长期耕作下 SOM 分子多样性变化
2. 分子热力学稳定性与热稳定性的协同关系
3. 分子多样性与热力学稳定性的长期权衡
4. 细菌丰富度对上述过程的调控作用

---

## 复现步骤
1.  安装依赖：`pip install numpy matplotlib scipy`
2.  运行 `01_fig1.py`：复现耕作处理下的箱线图（分子多样性、ΔGcox、热稳定性指数）
3.  运行 `02_fig2.py`：复现 SOM 热稳定性、分子多样性与向量角度的相关性分析
4.  运行 `03_fig3.py`：复现细菌丰富度与 SOM 分子特征的相关性分析
5.  运行 `04_fig4.py`：复现验证实验中不同时间点的分子特征变化
